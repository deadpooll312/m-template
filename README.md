# Матби
Верстка для web сайта Матби


### Структура проекта:

```text
|- dist (Мин. стили, картинки и js'ки)
|- src (HTML'ки и исходники CSS/JS находятся тут)
   |- assets
      |- css
      |- fonts
      |- img
      |- js
   |- **/*.html
|- .gitignore
|- gulpfile.js
|- package.json
|- README.md
```

В проекте не использовал SCSS/LESS.

```text
~$ node --version

```
